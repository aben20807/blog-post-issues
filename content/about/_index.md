# 關於我

我是黃柏瑄，沒啥綽號大家平常就都直接叫我柏瑄，最常用的表情符號是 OuO。目前是成功大學研究所 (實驗室: [ASRLab](https://chiaheng.wordpress.com/advanced-systems-research-lab/)) 的學生。目前自稱算是一個 vim 的重度使用者，本篇也是用 vim 編輯的 (最近...VSCode 真香)。家住彰化，有產葡萄、芒果、香蕉 (當然都不是我自己種的 OAO)。
